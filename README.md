# 2012
a GGJ 2012 game

Mayans were right. We are in trouble now and we need to defend our world. We are together or no!

Repeating "Q" and "W" keys generate money.
And click shoots.

If money is (< 10 m), you can't shoot. Press Q and W quickly and continuous.
